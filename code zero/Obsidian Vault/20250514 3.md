- [ ] 버전 관리
	- 버전 표기 방법: 시맨틱 버저닝(MAJOR.MINOR.PATCH) + 구분 태그(Tag)
		- MAJOR: 호환 불가
		- MINOR: 신규 기능 개발(FEATURE) 또는 기존 기능 삭제(DEPRECATED)
		- PATCH: 기존 기능 수정(CHANGE) 또는 오류 수정(FIXED)
		
	- 버전 관리 범위
		- iOS App
		- AOS App
		- Web
		- Back-end?
		
	- 버전 관리 요소
		- (필수) 버전
		- (필수) 날짜
		- (필수) 변경 이력 > 문체 논의 필요
		- 변경 이력 스크린샷 > 일반적으로 UI 변경 건만 특정
